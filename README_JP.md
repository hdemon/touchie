# Touchy

奇数サイズ等の画像を探し出すためのコマンドラインツールです。

## インストール

1. Ruby 1.8.7以上をインストールして下さい。
2. コンソールで以下のコマンドを実行して下さい。

```bash
$ gem install touchy
```

## 使い方

### 奇数サイズの画像を探し出す

```bash
touchy odd 画像のパス
```

#### 例

./hogeディレクトリ直下のpngファイルの中から探す。

```bash
touchy odd ./hoge/*.png
```

### 奇数サイズの画像を探し出し、偶数サイズに修正した上で、生じた隙間を透明なピクセルで埋める

```bash
touchy fix 画像のパス
```
